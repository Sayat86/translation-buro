Что такое апостиль

Официальное подтверждение документа для других стран

Апостиль — это специальный штамп, подтверждающий подлинность документа для использования за пределами Казахстана.

Апостиль необходим:

  • Для учебы за границей
  • Для работы за рубежом
  • Для эмиграции
  • Для подачи документов в иностранные учреждения
